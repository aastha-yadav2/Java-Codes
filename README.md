# ☕ Java Programming Practice Repository

This repository contains basic Java programs created for practice and learning.  
The programs demonstrate fundamental Java concepts such as conditional statements, arithmetic operations, loops, and user input using Scanner.

--------------------------------------------------

## 📂 Programs Included

### 1. Java Program — Input All Data Types Using Scanner
A program that takes input of different Java data types such as:
- Integer
- Float / Double
- Character
- String
- Boolean

This program demonstrates how to use the Scanner class for user input.

--------------------------------------------------

### 2. Addition and Subtraction
A Java program that takes two numbers as input and performs:
- Addition
- Subtraction

It displays the results of both operations.

--------------------------------------------------

### 3. Program for Calculating Gross Salary
This program calculates the Gross Salary of an employee using:
- Basic Salary
- HRA (House Rent Allowance)
- DA (Dearness Allowance)

Formula:
Gross Salary = Basic Salary + HRA + DA

--------------------------------------------------

### 4. Program to Check Leap Year
A Java program that checks whether a given year is a Leap Year or not.

Leap Year Conditions:
- Year must be divisible by 4
- If divisible by 100, it must also be divisible by 400

--------------------------------------------------

### 5. Program for Which Value is Greater
This program compares two numbers and determines which number is greater.

--------------------------------------------------

### 6. Program to Find Greatest of Three Numbers
A Java program that takes three numbers as input and finds the largest number among them using conditional statements.

--------------------------------------------------

### 7. Java Program to Check Number Categories (Table Format)

This program checks whether a number is:

- Even
- Odd
- Prime
- Composite
- Positive
- Negative

Example Output:

Number   Even   Odd   Prime   Composite   Positive   Negative
3        -      3     3       -           3          -

--------------------------------------------------

### 8. Application for Sick Leave
A simple Java program that prints a sick leave application format.

--------------------------------------------------
